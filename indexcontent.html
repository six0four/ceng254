  4. Quiz 7 will request the code to execute an SQL statement using JDBC which we may explore further in Lab 09. You may bring it as a print out including your username but you must leave the password string field blank ("") on your submission/print out. You are to target using NetBeans SE installed on the school computers with our Oracle Database. Chapter 2 of https://www.lynda.com/Java-tutorials/Java-Database-Integration-JDBC/110284-2.html will help you with this task.
  5. P.S. If installing on your own computer (which you probably already did in your Java course), start by downloading from https://netbeans.org/downloads/index.html
  6. If the installer cannot find a compatible JDK installation you probably want to Cancel the installation and install the following instead http://www.oracle.com/technetwork/java/javase/downloads/jdk-netbeans-jsp-142931.html
  7. To test your installation Choose File->New Project->Java->Java Application->Next->Finish.
  8. Add: System.out.println("Hello, World"); in to your main and choose Run.
  9. Download ojdbc6.jar and right-click on "Libraries" in your project navigator pane to add the JAR to your project. Using "import java.sql.*;" in your code will allow use of this.
  10. The examples use MySQL and HyperSQL, to build our connection string ("jdbc:oracle:thin:@apollo.humber.ca:1521:msit") explore right-clicking "Databases" in the project services pane to add the connection in a similar way to how it is in SQL Developer.
